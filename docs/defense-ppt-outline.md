## 答辩时的简短讲稿建议

### 1. 开场

各位老师好，我的毕业设计题目是《基于大语言模型的离散数学阶段性测试与个性化评估平台的设计与实现》。本次汇报主要从研究背景、系统设计、系统实现、测试结果和总结展望五个方面展开。

### 2. 架构页

本系统采用前后端分离单体架构，前端使用 Vue 3，后端使用 Spring Boot，数据库采用 MySQL 8，并通过兼容 OpenAI 协议的方式接入大语言模型。这样既保证了实现难度可控，也能满足毕业设计的功能要求。

### 3. AI 页

本项目中的 AI 主要用于三个环节：题目草稿生成、试卷草稿生成和主观题智能评阅。为了保证安全性与业务稳定性，所有模型调用都放在后端统一封装，而不是由前端直接访问第三方模型接口。

### 4. 测试页

在测试方面，除了常规自动化测试外，我还使用浏览器级自动化测试验证了真实的页面交互流程，包括教师组卷、学生作答、成绩查看和统计分析等功能，因此系统具备较好的可演示性和可用性。
