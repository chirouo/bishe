# 架构说明

## 目标

项目采用前后端分离单体架构，优先保证本科毕业设计的可实现性与可演示性，不引入 Redis、微服务、消息队列等额外复杂度。

## 后端

- `Spring Boot 2.7`
- `MyBatis-Plus`
- `MySQL 8`
- `JWT + 拦截器`

后端当前提供：

- 登录接口
- 当前用户信息接口
- 健康检查接口
- 数据库初始化脚本

## 前端

- `Vue 3`
- `Vue Router`
- `Pinia`
- `Element Plus`
- `Axios`
- `ECharts`

前端当前提供：

- 登录页
- 教师端基础布局
- 学生端基础布局
- 图表占位与页面骨架

## 业务模块规划

- 用户与角色
- 课程与知识点
- 题库管理
- 组卷管理
- 在线考试
- 自动判分
- 智能评阅
- 学习分析

## 自适应策略简化方案

首版不做复杂算法，采用规则驱动：

- 知识点正确率低于 60%，下次组卷提高该知识点权重
- 学生连续两次高分，减少基础题比例并增加中等题
- 将错题优先加入下次测试
